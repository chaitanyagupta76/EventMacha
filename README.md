# Event Macha

Production-ready React 18 application for creating event websites.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- i18next
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Project Structure

```
src/
├── components/      # Reusable UI components
├── sections/        # Page sections
├── pages/          # Route pages
├── data/           # JSON data files
├── i18n/           # Internationalization
├── App.jsx         # Main app component
└── main.jsx        # Entry point
```
